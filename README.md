### 阅读的方式
整体从上层往下层读，先读socket层，然后传输层，网络层，链路层。

1. 选好一个主题，并查询资料阅读该主题的相关背景
2. 大概浏览阅读相关源码
3. 仔细阅读源码实现原理
4. 最后再整理整个流程
